Что такое функция

В математике

— зависимость одной величины (значения функции) от другой (аргумента функции).

f(x) = 1 * 2x

  • f(1) = 2
  • f(3) = 8
  • f(0) = ?

Что такое функция

В Python

— фрагмент кода, который можно вызвать по имени.

Функции можно передать аргументы. Она может вернуть значение.

Функция map

Применяет какую-нибудь функцию ко всем элементам списка. Возвращает новый список из результатов работы этой фунции.

            strings = ["10", "35", "42"]
            numbers = map(int, strings)
  • Что будет записано в переменную numbers?
  • Какие аргументы принимает функция map?

Распаковка

Запись каждого элемента списка в отдельную переменную.

            numbers = [10, 35, 42]
            a, b, c = numbers
  • Что будет записано в переменные a, b и c?

Возвращаемся к задаче

            # Считываем строку, введенную пользователем "3 453 23"
            string = input()
            # Разбиваем строку на список строк ["3", "453", "23"]
            pieces = string.split()
            # Превращаем список строк в список чисел [3, 453, 23]
            numbers = map(int, pieces)
            # Распаковываем список в отдельные переменные
            a, b, c = numbers

Возвращаемся к задаче

Все эти четыре действия можно записать в одну строку без дополнительных переменных.

            a, b, c = map(int, input().split())
